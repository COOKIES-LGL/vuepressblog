### eslint 实用笔记

#### globals 字段值为一个对象，‌ 键为全局变量名，‌ 值为该变量是否允许被重写

```json
{
    "globals": {
        "jQuery": "readonly",
        "Vue": "writable"
    },
    "parserOptions": {
        "sourceType": "module", // 指定源代码的类型，‌'module'表示ES6模块
        "ecmaVersion": 2017, // 配置支持es2017的语法 async await 等
        "ecmaFeatures": {
            "jsx": true // 如果使用JSX，‌则设置为true
        }
    }
  }
}
```

