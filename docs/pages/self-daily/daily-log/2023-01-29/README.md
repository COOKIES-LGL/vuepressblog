本来还准备了一堆小游戏的20多个 只玩了6个就被姐夫家的人冲上了楼，真应该再多叫点人堵在楼梯口。

后知后觉 姐姐 已经不再只是我的姐姐了。  
她已为人妻，  
她已为人母，  
有了自己“好”字已成的家。  

小时候她会抓萤火虫放我拉链口袋里。  
小学她五年级，我学前班，  
我不去上学她总是用棍子赶着我去。  
初中她总会从镇里带一块钱三包的零食给我。  
后来高中大学，从言语中知道她受了很多委屈都在自己扛。  

一定一定要狠狠地幸福。
